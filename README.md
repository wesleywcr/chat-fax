# 🏃Getting Started

```sh
#  Clone Repository
$ git clone https://github.com/wesleywcr/chat-fax.git

# Go to the project folder
$ cd chat-fax
# Install Dependencies
$ yarn install
# Start application
$ npx expo start
```
